CREATING A PATH

drag the animal from inventory
Right Click -> Edit
Right Click -> Touch (the menu opens)
        
The Menu:

Record          Records the current position
Clear             Clears all saved positions
Clear Action   Clears the current "action"


Speed +        Increase speed
Speed -         Decrease speed

Pause      +    Increase time to Pause at this way point
Pause      -     Decrease time to Pause at this way point

Start              Start following the path

Move the animal to the place your path will begin
set the speed if desired
set a behavior if desired  (Pause, Feed Down, Feed Up) Behaviors vary from animal to animal
press Record

Move the animal to the next location
set the speed if desired
set a behavior if desired  (Pause, Feed Down, Feed Up) Behaviors vary from animal to animal
press Record

continue moving and recording, ending near the point you began

Press START to start the animal swimming
