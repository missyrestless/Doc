# AVsitter Custom Sequenced Animations

These folders contains `AVpos` and `[AV]sequence_settings` notecards for use
with the `AVsitter2` plugin `[AV]sequence` to provide animation sequences for
several vendor's dialog menus.

Animation sequenced `AVsitter` customization allows greater freedom of personal
interaction, eliminating the need to navigate dialog menus during moments of intimacy.

## [AV]sequence script and customized notecards

Most vendors do not include the `[AV]sequence` script or its configuration notecard
`[AV]sequence_settings`. This is an optional `AVsitter` plugin that enables sequences
of animations to be played when a single dialog menu button is clicked. Vendors do not
include the AVsitter sequence plugin as it consumes additional memory and they do not
want to push the limits for their production releases. However, most furniture can be
customized with animation sequences without issue. The vendors are just playing it safe.

You can download the `[AV]sequence` plugin script here:

```bash
wget -q -O '[AV]sequence' https://raw.githubusercontent.com/AVsitter/AVsitter/refs/heads/master/AVsitter2/Plugins/AVsequence/%5BAV%5Dsequence.lsl
```

Each of the folders here contains a customized `[AV]sequence_settings` notecard
and a customized `AVpos` notecard specifically tailored for the product.
