## Setup

**[Note:]** The Nerenzo little hangout scripts and notecards are not in the root prim.
It is necessary to edit the linked prims in the hangout to perform the customization.
Follow the detailed steps below.

To configure a Nerenzo little hangout with AVsitter sequences:

- Right click the hangout and select `Edit`
- Select the Contents tab in the Edit window
- Click the `Edit linked` checkbox
- Using `Ctrl .` (press both the Ctrl key and the Period key) locate the scripts and notecards
- Copy the `[AV]sequence_settings` notecard into the Contents tab of this linked prim
- Copy the `[AV]sequence` plugin script into the Contents tab of this linked prim
- Replace the `AVpos` notecard in the Contents tab with the `AVpos` notecard found in this folder
- Close the `Edit` window
- Right click the hangout and select `More` -> `More` -> `Scripts` -> `Reset Scripts`
- Wait for the scripts to reset (this takes a few minutes)

Sit on the hangout and click one of the main menu entries, e.g. click `SINGLE` or `FOREPLAY`.

You should see a menu entry of the form `SEQ-...`. Click this menu entry and your
avatar will cycle through all the animations in that submenu. Cycle times are currently
set to 60 seconds with menu arrows to step to the next animation, previous animation,
or pause animation sequencing.
